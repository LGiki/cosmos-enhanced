一个增强小宇宙播客网页端使用体验的浏览器插件，可以方便地下载小宇宙节目音频、下载小宇宙单集封面高清大图等。

# Features

- 下载
  - 下载单集音频
  - 下载单集封面原图
  - 下载播客封面原图
  - 下载主播头像原图
- 搜索
  - 在 ListenNotes 搜索播客
  - 在 ListenNotes 搜索单集
- 为播放器增加倍速功能
