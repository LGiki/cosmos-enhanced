# Cosmos Enhanced

<img src="./icons/icon-128.png" width="100" height="100" align="right" />

![GitHub](https://img.shields.io/github/license/lgiki/cosmos-enhanced?style=flat-square)

一个增强小宇宙播客页面使用体验的浏览器插件。

# Features

- 下载
  - 下载单集音频
  - 下载单集封面原图
  - 下载播客封面原图
  - 下载主播头像原图
- 搜索
  - 在 ListenNotes 搜索播客
  - 在 ListenNotes 搜索单集
- 为播放器增加倍速功能
